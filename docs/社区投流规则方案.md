# 旅行App「发现」社区投流规则方案

> 文档版本：v1.0
> 适用产品：旅行规划App（HTML移动端）「发现」社区板块
> 适用周期：以月为滚动单位（首期4周启动）
> 总预算基准：50,000元/月
> 覆盖城市：杭州、上海、成都、厦门、北京、西安

---

## 1. 投流目标

### 1.1 定量目标（月度）

| 目标维度 | 当前基线 | 月度目标 | 季度目标 | 计算口径 |
| --- | --- | --- | --- | --- |
| 社区DAU | 3,000 | 8,000 | 20,000 | 进入「发现」Tab的UV去重 |
| 社区渗透率 | 18% | 35% | 50% | 社区DAU / App整体DAU |
| 模板套用量（copies） | 1,800/月 | 6,000/月 | 18,000/月 | 用户点击"套用模板"并生成行程的去重次数 |
| 模板收藏量（favorites） | 1,200/月 | 4,500/月 | 12,000/月 | 模板卡片收藏按钮点击 |
| 模板点赞量（likes） | 3,500/月 | 12,000/月 | 35,000/月 | 模板卡片点赞按钮点击 |
| UGC产出量 | 200/月 | 800/月 | 3,000/月 | 用户自主发布/套用后改造的行程模板数 |
| 投流获客成本（CAC） | - | ≤18元 | ≤15元 | 投放总费用 / 新注册并访问社区用户数 |
| 模板到行程转化率 | 12% | 25% | 32% | 套用模板成功生成行程 / 模板详情页PV |

### 1.2 定性目标

1. **品类心智建立**：在用户认知中将"旅行规划=本App"，"找行程模板=来发现社区"，特别是在杭州、成都、厦门三个高潜力城市的搜索心智占位。
2. **内容供给质量提升**：建立"模板质量分"机制，使带 dayPlans 完整日程、含真实地址与 duration 的优质模板占比从 35% 提升至 65%。
3. **季节性匹配**：保证当季模板占比 ≥60%（如 7-8 月以避暑、海滨、亲子暑假为主，9-10 月以赏秋、CityWalk 为主）。
4. **KOL/KOC 资源池**：建立 50 人以上的旅行类创作者合作池，覆盖 6 个核心城市。
5. **品牌信任度**：通过专业攻略内容沉淀，使用户对社区模板的"靠谱感"评分（5分制）从 3.4 提升至 4.2。

## 3. 投放渠道选择

### 3.1 渠道主次划分

| 层级 | 渠道 | 定位 | 预算占比 | 适配人群 |
| --- | --- | --- | --- | --- |
| 主渠道1 | 小红书 | 种草+模板转化核心阵地 | 38% | P1、P3、P2 |
| 主渠道2 | 抖音 | 短视频爆款+拉新引流 | 28% | P2、P1 |
| 次渠道1 | 微信公众号/视频号 | 信任沉淀+亲子深度决策 | 15% | P3、P4 |
| 次渠道2 | B站 | 长视频深度攻略+品牌背书 | 10% | P4、P2 |
| 辅助渠道 | 微博 | 话题热搜+KOL扩散 | 6% | 全人群 |
| 辅助渠道 | 知乎 | SEO+专业问答占位 | 3% | P4 |

### 3.2 各渠道适配内容形式与理由

#### 3.2.1 小红书（主渠道1）

- **适配内容形式**：
  - 图文笔记（9图+长文）：单城市2-3日行程攻略，封面为行程地图或景点拼图
  - 视频笔记（30-60秒）：景点快剪+机位教学
  - 合集笔记：6城避暑/CityWalk/亲子等主题合集
- **理由**：
  - 用户主动搜索"XX城市旅游攻略"意图强，与社区模板套用场景高度契合
  - 女性用户占比高，与P1白领、P3亲子人群重合
  - 长尾流量好，一篇优质笔记3个月内持续带量
  - 模板卡片瀑布流形态可直接复用为小红书图文素材

#### 3.2.2 抖音（主渠道2）

- **适配内容形式**：
  - 短视频（15-45秒）：单景点机位/单日CityWalk路线/避坑提醒
  - 直播：周末达人带看+模板现场套用
  - 信息流广告：程流卡片跳转App下载/模板详情
- **理由**：
  - 流量池大，爆款可短期撬动10万+曝光
  - 算法分发适合P2 Z世代拉新
  - 视频形式能直观展示 dayPlans 中的景点串联节奏

#### 3.2.3 微信公众号/视频号（次渠道1）

- **适配内容形式**：
  - 长图文：5日深度行程（含每日 spots、餐厅、避坑），底部挂模板套用入口
  - 视频号短视频：1-3分钟攻略
  - 社群裂变：宝妈群/同城群模板分享
- **理由**：
  - 私域信任度高，P3亲子人群决策周期长、需深度内容
  - 公众号文章可承载 dayPlans 完整结构，与社区模板字段一一对应
  - 视频号近微信生态，便于社交裂变

#### 3.2.4 B站（次渠道2）

- **适配内容形式**：
  - 中长视频（5-15分钟）：单城市深度Vlog+行程解读
  - 知识区投稿：旅行规划方法论、避坑指南
  - 互动视频：用户选择路线分支
- **理由**：
  - P4资深用户与P2学生用户活跃，对深度内容容忍度高
  - 弹幕沉淀形成口碑背书
  - 视频可长期被搜索复用

#### 3.2.5 微博（辅助渠道）

- **适配内容形式**：
  - 话题营销：#周末去哪儿# #杭州2日游# #带娃旅行不踩雷#
  - KOL转发+抽奖：转发套用模板截图抽奖
- **理由**：
  - 适合事件营销与热搜放大
  - 与KOL/KOC联动扩散效率高

#### 3.2.6 知乎（辅助渠道）

- **适配内容形式**：
  - 长回答：「XX城市旅游有哪些必备攻略」「带3岁娃去北京怎么安排」
  - 专栏文章：行程方法论
- **理由**：
  - SEO权重高，长期被动流量
  - 决策阶段用户多，转化质量好

### 4.3 预算使用规则

1. **内容制作费**：仅用于投流专用素材（拍摄、剪辑、设计），不含常规运营内容。
2. **投放推广费**：信息流/竞价广告费，按周监控ROI，单渠道连续2周低于基准ROI 0.6 即暂停。
3. **KOL/KOC合作费**：单篇笔记/视频合作单价控制在以下区间：
   - KOC（粉丝1k-1w）：300-800元/篇
   - 腰部KOL（粉丝1w-10w）：1,500-5,000元/篇
   - 头部KOL（粉丝10w+）：8,000-15,000元/篇（每月不超过2篇）
4. **工具/服务费**：小红书蒲公英/抖音星图平台服务费等。
5. **预留机动**：每月从总预算中预提 5%（2,500元）作为爆款追投预备金，月末未使用结转次月。

### 5.5 素材规范

| 素材类型 | 规范要求 |
| --- | --- |
| 封面图 | 比例3:4（小红书）或9:16（抖音/视频号）；含城市名+天数+人群标签；不堆砌文字 |
| 配图 | 单图不低于1080px；保留景点原貌不做过度滤镜；含地图标注优先 |
| 视频 | 横屏1920×1080或竖屏1080×1920；H.264编码；MP4；码率≥8Mbps；字幕必加 |
| 文案 | 首行含核心钩子（≤20字）；含结构化要点（人数/天数/人均/季节）；末尾含行动指令"评论区领模板" |
| 模板入口 | 每条内容必须挂1个社区模板跳转链接或口令，便于追踪转化 |
| 合规 | 不使用未授权肖像；不夸大宣传；涉景点门票/开放时间需注明来源与时效 |

## 7. 效果评估指标

### 7.1 核心KPI

| KPI | 月度目标 | 计算公式 | 数据来源 |
| --- | --- | --- | --- |
| 社区DAU | 8,000 | 当日访问「发现」Tab的去重用户数 | App埋点 |
| 模板套用量（copies） | 6,000 | 套用模板并生成行程的去重次数 | App埋点 |
| UGC产出量 | 800 | 用户自主发布/改造模板数 | App后台 |

| 社区次留 | 30% | 次日再次访问社区用户 / 当日社区新增用户 | App埋点 |

### 7.2 过程指标

| 指标 | 目标 | 说明 |
| --- | --- | --- |
| 曝光量 | ≥300万 | 各渠道曝光总和 |
| 点击率（CTR） | ≥3.5% | 信息流广告点击/曝光 |
| 互动率 | ≥6% | 点赞+评论+收藏 / 曝光 |
| 模板详情页PV | ≥50,000 | 社区模板卡片点击 |
| 笔记/视频爆文率 | ≥8% | 单条互动量超该渠道均值3倍视为爆文 |
| KOL/KOC履约率 | 100% | 按时按规发布 |
| 单篇笔记带量 | ≥80次套用 | 单篇内容带来的模板套用次数 |

### 7.3 渠道ROI计算公式

```
渠道ROI = （该渠道带来的模板套用次数 × 单次套用LTV估值） / 该渠道总投入

其中：
- 单次套用LTV估值 = 套用模板用户的人均6个月留存贡献毛利 / 平均套用次数
- 首期估值：单次套用LTV = 12元（保守值，后续按真实数据校准）

示例：
小红书渠道投入 19,000 元，带来模板套用 2,200 次
ROI = (2,200 × 12) / 19,000 = 1.39
判定：ROI ≥1.0 为合格，≥1.5 为优秀，<0.6 为低效
```

## 8. 优化机制

### 8.1 A/B测试方法

| 测试维度 | 变量 | 对照设置 | 持续时长 | 判定指标 |
| --- | --- | --- | --- | --- |
| 封面图 | 人物vs风景vs地图 | 同一选题两版封面 | 3天 | 点击率 |
| 标题 | 痛点型vs利益型vs悬念型 | 同一内容3版标题 | 3天 | 点击率 |
| 行动指令 | "套用模板"vs"领同款行程"vs"评论区领" | 同一笔记不同CTA | 5天 | 套用次数 |
| 投放时段 | 工作日午间vs晚间vs周末 | 同一素材不同时段 | 5天 | CTR+套用 |
| KOL层级 | 腰部1篇vs KOC 5篇 | 等预算对照 | 7天 | 模板套用总量 |
| 落地页 | 模板详情页vs城市聚合页 | 同一素材不同落地 | 7天 | 套用转化率 |

**执行原则**：单次只变1个变量；样本量不低于1,000次曝光；显著性差异≥10%方可定论。

### 8.2 周复盘机制

| 复盘项 | 频次 | 参与人 | 产出 |
| --- | --- | --- | --- |
| 日数据快报 | 每日 | 运营 | 飞书日报 |
| 周复盘会 | 每周一 | 运营+内容+投放 | 周报复盘文档 |
| 爆文拆解 | 随时 | 内容策划 | 爆文拆解卡 |
| 渠道ROI评估 | 每周 | 投放 | 渠道ROI榜 |
| 月度战略复盘 | 每月末 | 项目负责人+全员 | 月报+次月方案 |

**周复盘必答5问**：
1. 本周哪3篇内容互动/转化最好？原因是什么？
2. 哪个渠道ROI低于0.6？是否进入观察/淘汰？
3. 哪些城市/分类Tab的模板供给不足？
4. UGC活动参与度如何？是否需要调整激励？
5. 下周选题Top 5是什么？

### 8.3 低效渠道淘汰标准

| 评估项 | 阈值 | 处理 |
| --- | --- | --- |
| 渠道ROI | 连续2周 <0.6 | 暂停投放，转入观察 |
| 单渠道CTR | 连续2周 <1.5% | 调整素材后重测1周，仍不达标则暂停 |
| 单渠道CAC | 连续2周 >25元 | 缩减预算50%并复盘 |
| KOL单篇带量 | <30次套用 | 不再续约，纳入观察池 |
| 单篇笔记曝光 | <500且互动<20 | 不再追投，素材归档为负面样本 |

> 淘汰不等于永久放弃：被淘汰渠道/素材进入"休眠池"，每季度评估1次是否重启。

### 8.4 内容迭代方向

| 迭代方向 | 触发条件 | 行动 |
| --- | --- | --- |
| 城市供给扩充 | 某城市搜索量周增≥30%但模板<5个 | 2周内补足10个模板 |
| 季节性切换 | 进入新季节前2周 | 上线当季专属主题合集，旧季模板加"季节限定"标签 |
| 爆文复制 | 单篇爆文（互动超均值3倍） | 1周内复刻同选题不同城市版本≥3篇 |
| UGC激活 | UGC产出量连续2周下滑 | 加码UGC激励（流量扶持+实物奖励） |
| 模板质量提升 | 模板到行程转化率<20% | 清理无dayPlans或仅1日模板，下架低质模板 |
| 分类Tab冷热 | "小众""自然"Tab PV<5% | 增加相关主题投流，调整首页Tab排序权重 |
