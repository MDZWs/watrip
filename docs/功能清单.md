## 一、页面结构（index.html）

### 主要页面（.page）
| 页面ID | 页面名称 | 说明 |
|---|---|---|
| homePage | 首页 | 问候语、统计卡、快捷组件、快捷入口 |
| tripsPage | 我的行程 | 行程列表 + 旅行记忆双Tab |
| inputPage | AI规划输入页 | 目的地输入、日期、偏好设置 |
| manualPlanPage | 手动规划页 | 社区推荐、筛选、景点选择 |
| tripDetailPage | 行程详情页 | 行程信息、列表/地图模式、记账、行李 |
| communityPage | 社区发现页 | 搜索、分类、瀑布流 |
| templatePreviewPage | 模板预览页 | 行程概述、列表/地图模式 |
| profilePage | 个人中心 | 头像、积分、菜单 |

- `expenseAddOverlay` - 记账添加弹窗
