## 4. 优先级与建议

### 必须先做（阻塞性）
- Phase 1.1 → 1.2 → 1.3：地基，所有功能依赖
- Phase 2.1：有内容才有社区

### 核心价值（MVP 闭环）
- 里程碑二（专栏）+ 里程碑三（积分）：形成"发布→被看→赚积分→升级"闭环

### 差异化亮点
- 里程碑四（吐槽打卡）：地点维度的真实内容，区别于普通社区

### 锦上添花
- 里程碑五（游戏化）：提升留存与活跃，但不阻塞核心功能

## 5. 风险与应对

| 风险 | 应对 |
|------|------|
| Supabase 国内访问慢 | 初版验证产品，后续自托管或加速 |
| 用户上传图片审核成本 | 第一阶段只用高德图，第二阶段再开上传 |
| 内容冷启动（无 UGC） | 用现有种子模板 + 官方图文游记做种子内容 |
| 积分防刷难度大 | UV 去重 + 频率限制 + 异常监控 |
| 人工审核人力不足 | 先靠自动关键词过滤 + 举报触发，复审延后处理 |
