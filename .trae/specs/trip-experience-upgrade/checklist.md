# 行程体验升级 - 验收清单

## 基础功能
- [ ] 行程列表只有「旅行中」「回忆长廊」两个Tab
- [ ] 已开始的行程在「旅行中」Tab置顶
- [ ] 未开始行程按出发日期升序排列
- [ ] 已完成行程在「回忆长廊」按完成时间倒序
- [ ] 卡片右上角显示📍旅行中或✅已完成标识
- [ ] 卡片内天数预览区域可上下滚动
- [ ] 详情页返回列表时数据自动刷新
- [ ] 修改时间/景点/花费后，卡片信息同步更新

## 照片功能
- [ ] 默认显示POI官方封面图（若有）
- [ ] 相机按钮在照片右上角，点击可选拍照/相册
- [ ] 上传1张：全宽作为封面
- [ ] 上传2张：左右并排
- [ ] 上传3张：一主两副布局
- [ ] 最多支持3张，超出提示
- [ ] 点击照片可全屏预览，滑动切换

## 交互功能
- [ ] 长按景点卡片可拖拽排序
- [ ] 拖拽时卡片有浮起阴影效果
- [ ] 支持跨天移动景点
- [ ] 两个景点之间有+号按钮，可插入新景点
- [ ] 拖拽到底部删除区域可删除景点
- [ ] 原删除按钮保留可用
- [ ] 拖拽完成后时间自动重算
- [ ] 「到此一游」按钮已移除

## 写想法（手帐）功能
- [ ] 未记录的景点显示红点角标
- [ ] 设置中可开关红点提示
- [ ] 点击红点打开底部编辑面板
- [ ] 支持「实况打卡」「回忆补记」两种模式切换
- [ ] 实况模式自动定位校验位置
- [ ] 支持加粗、斜体格式
- [ ] 滑动条可调整字体大小
- [ ] 支持段落对齐、缩进
- [ ] 可一键插入当前时间
- [ ] 可插入景点照片到文字中，实现图文混排
- [ ] 每30秒自动保存，内容不丢失
- [ ] 多次编辑内容正确保存

## AI总结与分享
- [ ] 编辑器底部有「AI总结成攻略」按钮
- [ ] 点击后AI将随笔整理成结构化攻略
- [ ] 总结结果可编辑修改
- [ ] 可一键分享到社区，自动填充游记内容

## 数据结构
- [ ] 所有数据结构化存储
- [ ] 手帐内容为标准HTML格式
- [ ] 照片为base64存储，可离线访问
- [ ] 提供tripToHtml方法，可生成完整行程HTML
- [ ] 无控制台报错
